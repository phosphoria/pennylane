Outil interne · gratuit · sans installation compliquée

Complétez automatiquement les coordonnées de vos fournisseurs Pennylane

À partir du numéro de TVA (ou du nom), l'outil retrouve le SIRET, l'adresse, le code postal, la ville et le pays manquants — puis prépare un fichier prêt à réimporter dans Pennylane.

À quoi ça sert

Pennylane permet d'exporter les fiches incomplètes en CSV, puis de réimporter le fichier corrigé. L'outil automatise l'étape intermédiaire.

🇫🇷 Fournisseurs français

SIRET, adresse, code postal et ville récupérés via la base officielle Sirene/INSEE.

🇪🇺 Fournisseurs européens

Nom et adresse via VIES (Commission européenne), quand le pays les transmet.

🔎 Sans numéro de TVA

Recherche prudente par nom : complété seulement en cas de correspondance unique et certaine.

✅ Prêt à réimporter

Un fichier « traités » réimportable dans Pennylane + un fichier « à traiter » + un rapport.

Installation

Pas de ligne de commande. À faire une seule fois — suivez les images.

  1. Télécharger l'outil Cliquez sur le bouton « ⬇ Télécharger » en haut à droite de cette page (alternative : la dernière version sur GitHub).
  2. Décompresser & ranger Double-cliquez sur le fichier .zip téléchargé. Vous obtenez un dossier avec deux fichiers ; gardez-les ensemble. Important : déplacez ce dossier dans votre dossier personnel (l'icône maison à votre nom dans le Finder) — évitez de le laisser dans Téléchargements, Bureau ou Documents, que macOS protège (sinon l'outil ne pourra pas lire les fichiers).
  3. Autoriser l'outil (une seule fois) Double-cliquez sur Traiter un export Pennylane.command. macOS bloque par sécurité tout fichier venu d'internet : il faut l'autoriser une fois, en 3 écrans.
  4. Python (si macOS le demande) Python est simplement le moteur gratuit et standard qui exécute le script (édité par une fondation à but non lucratif, utilisé partout dans le monde) — rien d'inquiétant. Si une fenêtre propose de l'installer, cliquez « Installer », patientez, puis relancez l'outil.
💡 macOS uniquement. Le blocage de sécurité est normal pour tout fichier téléchargé : la manipulation des 3 écrans ne se fait qu'une seule fois.

Utilisation au quotidien

Une fois la fenêtre ouverte, tout se fait au glisser-déposer.

  1. Exporter depuis PennylaneRécupérez l'export CSV « fournisseurs » (ou « clients ») à compléter.
  2. Glisser-déposerGlissez ce fichier CSV (ou le dossier d'un client) dans la fenêtre, puis appuyez sur Entrée.
  3. Récupérer les résultatsTrois fichiers sont créés à côté de l'original : …traites.csv (à réimporter), …a_traiter.csv (à finir à la main) et …report.txt (le détail, avec liens de vérification).
  4. RéimporterImportez le fichier …traites.csv dans Pennylane.

Limites connues

Quelques cas que l'outil ne peut pas compléter tout seul — à finir à la main.

  • Certains pays de l'UE ne communiquent pas l'adresse (Allemagne, Espagne, Pays-Bas, Italie…). Le pays est corrigé, mais l'adresse reste à saisir.
  • Fournisseurs hors Union européenne (Royaume-Uni, États-Unis…) : seul le pays est renseigné, pas l'adresse.
  • Entreprises individuelles « protégées » : la loi les autorise à ne pas publier leur adresse — à compléter manuellement.
  • macOS uniquement pour le lanceur en double-clic.
